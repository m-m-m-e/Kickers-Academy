# Kickers tech Stack
Project Kickers is a custom team website with a unified admin system and a store that the admin can fully control. The stack should support the website now and a mobile app later without needing to rebuild the core system.

# Chosen Stack

- Frontend website: Next.js - fast, good SEO and clean for the future app.
- Language: TypeScript - safe writing and debugging.
- UI styling: Tailwind CSS - consistency.
- Reusable UI components: shadcn/ui - consistency.
- Backend API: NestJS - bs logic and authentication.
- Database: PostgreSQL - structured data.
- Database access layer: Prisma - clear data and query management.
- Authentication and roles: Auth.js - Access control.
- Payments: Stripe - custom store.
- Media storage: Cloudinary - media.
- Analytics: PostHog - analysis.




# Target achievement

- A public team website
- A full admin dashboard
- Content management for all of the pages
- A custom merchandise store
- Order management
- Role-based access for admins
- Analytics and usage insights
- A future mobile app that can reuse the same backend
