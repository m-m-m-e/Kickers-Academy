# Theme, Appearance, and Layout

- Primary theme: Dark 
- Main colors: Black, red, and white

# Color System

# Primary Colors
- Black: main background and brand base
- Red: highlights, buttons, active states, and key calls to action
- White: text, contrast, and clean space

# Supporting Colors
- Dark gray for panels and sections
- Soft gray for borders and subtle text
- Muted red for hover states and accents

# Public Website Style

The user-facing side feel

# Visual Traits
- Large hero banners
- Full-width image sections
- Strong overlays and gradients
- Featured content cards
- Match-day style layouts
- Store highlights presentation

# Layout Style
- Top navigation with clear menu sections
- Hero section with team identity and featured message
- Featured modules below the hero
- Story-driven sections for news, events, and team information
- Gallery and store sections with strong visuals
- Clear call-to-action buttons for join, donate, and shop


# Admin Dashboard Style
Admin facing side

# Visual Traits
- Dark sidebar navigation
- Card-based dashboard layout
- Clean tables for products, posts, events, and orders
- Clear metrics and charts
- Strong focus on usability

# Layout Style
- Left sidebar for modules
- Top bar for notifications, search, and account controls
- Main dashboard area with summary cards
- Dedicated module pages for CRUD management
- Analytics panels for views, sales, and activity


# Typography Direction

- Use bold, modern headings
- Use clean readable body text
- Keep spacing generous enough for a premium look

# Component Style

- Rounded but sharp-feeling cards
- Bold buttons with clear hover states
- Prominent banners and section headers
- Dark panels with red accents


# Final Direction

The final result should feel like a real football club platform.
